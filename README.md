# Game32

## Overview

Game32 is a bullet hell shoot 'em up video game. Player need to control the character to elude bullets while defeating the boss by shooting bullets.

## Control
→ ↑ ↓ ←: Movement
Space: Shoot Bullet
Z: Speed control
C: Bomb
ESC: exit/pause

## Dependencies

- **OpenGL (>= 3.3)**
- **GLUT (FreeGLUT, >= 3.0.0)**
- **CMake (>= 3.12)**

## How to run

```
cd build
cmake ..
./Game32
```
